# Steadily

A habit tracker with opinions about design. Custom themes, smooth animations, and a subscription model that unlocks the aesthetic options — because if you're going to stare at a habit tracker every day, it should look good.

## Features

- **Clean habit dashboard** with daily/weekly/monthly views
- **Theme system** — multiple color palettes and visual styles
- **Streak visualization** with heatmap calendar
- **Flexible recurrence** — daily, specific days, intervals
- **StoreKit 2 subscriptions** unlocking premium themes
- **Haptic feedback** on completions
- **Dark mode** with theme-aware gradients

## Tech Stack

- SwiftUI
- StoreKit 2
- SwiftData
- Combine

## Getting Started

```bash
git clone https://github.com/sebastiandoyle/steadily.git
cd steadily
open *.xcodeproj
```

Requires Xcode 15+ and iOS 17+.
